- **写产品文章时，必须从GitHub仓库/官网获取真实产品截图，嵌入文章中**
- 不要只用AI生成封面和配图，真实产品截图更有说服力
- 具体做法：
  1. `git clone` 产品仓库后，搜索 assets/、docs/img/、screenshots/ 等目录
  2. 下载官网产品截图
  3. 用 Playwright 截取官网/Demo页面
  4. 将真实截图嵌入文章markdown中
  5. 封面优先用真实截图，AI生图作为备选
- **这个规则必须遵守，不要再犯同样的错误！**
